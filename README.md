# SoloSophy Web App Prototype

SoloSophy — The Art of Investment

## Current status

This repository now contains the first static prototype of the SoloSophy market intelligence web app.

## Files

- `index.html` — app shell and page structure
- `styles.css` — SoloSophy visual design system
- `app.js` — language toggle, market cards, news filter, scenarios

## Current capabilities

- Arabic-first interface
- English language toggle
- Market board placeholders for Gold, Silver, Oil and Equities
- News cards prepared for API integration
- Asset filter for Gold, Silver, Oil and Equities
- Scenario panel with probability bars
- Dubai UTC+4 timestamp logic
- Mobile responsive layout
